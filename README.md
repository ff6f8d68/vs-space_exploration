vs space Exploration is a space addon for vs2.
Like kontraption but with more fetures,
Like The Lightspeed engine,
Or that real space travel is intigrated

__This mod's dependencies include:__

- Valkyrien Skies 2
- VS Core
- Kotlin for Forge
